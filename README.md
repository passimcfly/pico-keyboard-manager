![screenshot](./gitImages/screenshot.jpg)

## Get Started

### 1. clone this repository

```bash
git clone https://github.com/passimcfly/pico-keyboard-manager.git
```

### 2. install packages

```bash
npm i
```

### 3. Run

### dev mode

```bash
npm run app:dev
```

### preview mode

```bash
npm run app:preview
```

### build app

```bash
npm run app:build
```
